Über die Verschlüsselung

Die Verschlüsselung des Adressbuchs basiert auf AES128. Der Einfachheit halber wird ein statisches Salt verwendet.Die Anzahl der Runden zur Verschlüsselung beträgt 65535.

Adressbuch verschlüsselt speichern

Um ein Adressbuch verschlüsselt abspeichern zu können muss ein Adressbuch in der Anwendung geöffnet sein. Anschließend geht man wie folgt vor:

  1. Sicheres Passwort wählen und 2x eingeben
  2. Pfad wählen in welchem die verschlüsselte Datei abgelegt werden soll
  3. Bestätigen

Verschlüsseltes Adressbuch öffnen

Um ein verschlüsseltes Adressbuch zu öffnen geht man wie folgt vor:

  1. Passwort für die Entschlüsselung angeben
  2. Pfad zur verschlüsselten Datei angeben
  3. Pfad angeben in welchem die entschlüsselte Datei abgelegt werden soll
  4. Bestätigen

Kontakt-Verwaltung

Um Kontakte anlegen zu können muss entweder ein Adressbuch geöffnet oder erstellt werden. Anschließend wird das Menü zur Kontaktverwaltung aktiviert und es können neue Kontakte angelegt und vorhandene gelöscht oder bearbeitet.
